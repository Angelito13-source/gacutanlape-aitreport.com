What is Edge Computing?

Edge computing is a distributed computing paradigm that brings computation and data storage closer to the sources of data.
This is expected to improve response times and save bandwidth. The term refers to an architecture rather than a specific technology.

Edge computing is an alternative architecture to cloud computing for applications that require high-speed and high availability.
